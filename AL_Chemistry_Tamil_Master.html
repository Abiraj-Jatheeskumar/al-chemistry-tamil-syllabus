அறிமுகம்

1. பாட அறிமுகம்

பாடத்தின் நோக்கம்

அணு என்றால் என்ன? அணுவின் உட்கட்டமைப்பு எவ்வாறு உள்ளது? இலத்திரன்கள் அணுவில் எங்கே, எவ்வாறு அமைந்துள்ளன? என்னும் அடிப்படை வினாக்களுக்கு விடையளிப்பதே இந்தப் பாடத்தின் நோக்கமாகும்.

முக்கிய கருத்து

- அணு மூன்று அடிப்படைத் துணிக்கைகளால் ஆனது: புரோட்டான் (p⁺), நியூட்ரான் (n⁰), இலத்திரன் (e⁻)

- இலத்திரன்கள் குறிப்பிட்ட ஆற்றல் மட்டங்களில் மட்டுமே இருக்கும்

- குவாண்டம் இயந்திரவியல் மாதிரி இலத்திரன்களின் நிகழ்தகவு அமைவை விவரிக்கிறது

ஏன் இந்தப் பாடம் முக்கியம்?

- அனைத்து இரசாயன வினைகளுக்கும் அணு அமைப்பே அடிப்படை

- பிணைப்பு கோட்பாட்டை புரிந்துகொள்ள இது அவசியம்

- தனிமங்களின் பண்புகளை விளக்க இது உதவுகிறது

2. முக்கிய கோட்பாடுகள்

2.1 அணுவின் வரலாற்று வளர்ச்சி

டால்டனின் அணுக் கோட்பாடு (1808)

- ஒவ்வொரு தனிமமும் அணு எனப்படும் சிறிய துகள்களால் ஆனது

- ஒரே தனிமத்தின் அணுக்கள் அனைத்தும் ஒரே மாதிரியாக இருக்கும்

- வேறுபட்ட தனிமங்களின் அணுக்கள் வேறுபட்டவை

- ❌ பலவீனம்: அணுவை பிரிக்க முடியாது என்றார் - இது தவறானது

தொம்சனின் அணு மாதிரி (1904)

- "பழச்செய்தி கேக்" (Plum Pudding) மாதிரி

- நேர்மின் சுமை கொண்ட கோளத்தில் எதிர்மின் சுமை கொண்ட இலத்திரன்கள் பரவி இருக்கும்

- J.J. தொம்சன் 1897 இல் இலத்திரனை கண்டுபிடித்தார்

- இலத்திரனின் திணிவு: 9.109 × 10⁻³¹ kg

- இலத்திரனின் ஏற்றம்: -1.602 × 10⁻¹⁹ C

ரூதர்போர்டின் அணு மாதிரி (1911)

- தங்க படலத்தில் ஆல்பா துகள்கள் எறிதல் சோதனை (Gold Foil Experiment)

- கண்டுபிடிப்புகள்:

- பெரும்பான்மையான ஆல்பா துகள்கள் நேரடியாக கடந்து சென்றன → அணுவில் காலி இடம் அதிகம்

- சிலவே திரும்பின → மையத்தில் மிகச்சிறிய, அடர்த்தியான, நேர்மின் சுமை கொண்ட கருவம் (nucleus) உள்ளது

- இடர்கள்:

- கருவைச் சுற்றும் இலத்திரன்களின் நிலைப்புத்தன்மையை விளக்க முடியவில்லை

- கதிர்வீச்சு ஏன் நீடிக்காது என்று விளக்க முடியவில்லை

போரின் அணு மாதிரி (1913)

- இலத்திரன்கள் குறிப்பிட்ட ஆற்றல் மட்டங்களில் (energy levels) மட்டுமே சுற்றும்

- இலத்திரன் ஒரு மட்டத்தில் சுற்றும்போது ஆற்றல் கதிர்வீச்சாக வெளியேறாது

- இலத்திரன் உயர் மட்டத்திலிருந்து தாழ் மட்டத்திற்கு தாவும்போது ஒளிக்கதிர் வெளியேறும்

- ஒவ்வொரு மட்டத்திலும் அதிகபட்ச இலத்திரன் எண்ணிக்கை: 2n² (n = மட்டம்)

ஆற்றல் மட்டங்கள்:
n=1 → K தோடு → அதிகபட்சம் 2 இலத்திரன்கள்
n=2 → L தோடு → அதிகபட்சம் 8 இலத்திரன்கள்
n=3 → M தோடு → அதிகபட்சம் 18 இலத்திரன்கள்
n=4 → N தோடு → அதிகபட்சம் 32 இலத்திரன்கள்

2.2 குவாண்டம் இயந்திரவியல் மாதிரி

ஹைசன்பர்கின் நிச்சயமின்மைக் கோட்பாடு

$$\Delta x \cdot \Delta p \geq \frac{h}{4\pi}$$

- Δx = நிலையின் நிச்சயமின்மை

- Δp = உந்தத்தின் நிச்சயமின்மை

- h = பிளாங்க் மாறிலி (6.626 × 10⁻³⁴ J·s)

முக்கிய கருத்து: இலத்திரனின் சரியான நிலையும் உந்தமும் ஒரே நேரத்தில் துல்லியமாக அறிய முடியாது.

ஷ்ரோடிங்கர் சமன்பாடு

- இலத்திரன்களை அலை செயல்பாடாக (wave function ψ) விவரிக்கிறது

- |ψ|² = ஒரு குறிப்பிட்ட இடத்தில் இலத்திரன் இருக்கும் நிகழ்தகவு

2.3 குவாண்டம் எண்கள் (Quantum Numbers)

நான்கு குவாண்டம் எண்கள் ஒவ்வொரு இலத்திரனையும் தனித்துவமாக அடையாளப்படுத்துகின்றன:

1. முதன்மை குவாண்டம் எண் (n) - Principal Quantum Number

- n = 1, 2, 3, 4, ... (நேர்முழு எண்கள்)

- ஆற்றல் மட்டத்தையும் கருவிலிருந்து தூரத்தையும் குறிக்கிறது

- n அதிகரிக்க ஆற்றலும் அதிகரிக்கும்

2. கோணவுந்த குவாண்டம் எண் (l) - Azimuthal Quantum Number

- l = 0, 1, 2, ..., (n-1)

- உதோட்பகுதியின் (orbital) வடிவத்தை தீர்மானிக்கும்

l மதிப்பு உதோட்பகுதி வடிவம்
0 s கோள வடிவம்
1 p டம்பல் வடிவம் (dumbbell)
2 d சிக்கலான வடிவம்
3 f மேலும் சிக்கலான வடிவம்

3. காந்த குவாண்டம் எண் (mₗ) - Magnetic Quantum Number

- mₗ = -l, -(l-1), ..., 0, ..., (l-1), l

- மொத்தம் (2l+1) மதிப்புகள்

- உதோட்பகுதியின் திசையை தீர்மானிக்கும்

உதோட்பகுதி l mₗ மதிப்புகள் உதோட்பகுதிகளின் எண்ணிக்கை
s 0 0 1
p 1 -1, 0, +1 3
d 2 -2,-1,0,+1,+2 5
f 3 -3,-2,-1,0,+1,+2,+3 7

4. சுழல் குவாண்டம் எண் (mₛ) - Spin Quantum Number

- mₛ = +½ அல்லது -½ மட்டுமே

- இலத்திரனின் சுழலை (↑ அல்லது ↓) குறிக்கும்

2.4 ஒளிர்வு நிழற்படை (Emission Spectrum)

ஹைட்ரஜன் அணுவின் ஒளிர்வு நிழற்படை

$$\frac{1}{\lambda} = R_H \left(\frac{1}{n_1^2} - \frac{1}{n_2^2}\right)$$

- λ = அலைநீளம்

- R_H = ரிட்பர்க் மாறிலி = 1.097 × 10⁷ m⁻¹

- n₁ = தாழ் மட்டம், n₂ = உயர் மட்டம் (n₂ > n₁)

நிழற்படைத் தொடர்கள்:

தொடர் n₁ n₂ கதிர்வீச்சு பகுதி
லைமன் (Lyman) 1 2,3,4,... புறவூதா (UV)
பால்மர் (Balmer) 2 3,4,5,... தெரியும் ஒளி
பாஷன் (Paschen) 3 4,5,6,... அகச்சிவப்பு (IR)
பிராக்கட் (Brackett) 4 5,6,7,... அகச்சிவப்பு (IR)
பஃபண்ட் (Pfund) 5 6,7,8,... அகச்சிவப்பு (IR)

2.5 உதோட்பகுதி (Orbital) வடிவங்கள்

s உதோட்பகுதி:

- கோள (sphere) வடிவம்

- n அதிகரிக்க அளவு அதிகரிக்கும்

- ஒரு உதோட்பகுதி மட்டுமே உண்டு

p உதோட்பகுதி:

- டம்பல் (dumbbell) வடிவம்

- மூன்று உதோட்பகுதிகள்: px, py, pz

- x, y, z அச்சுகளில் திசைகாட்டியாக இருக்கும்

d உதோட்பகுதி:

- சிக்கலான வடிவங்கள்

- ஐந்து உதோட்பகுதிகள்: dxy, dyz, dxz, dx²-y², dz²

2.6 இலத்திரன் நிரப்புவதற்கான விதிகள்

ஆஃப்பாவ் கோட்பாடு (Aufbau Principle)

- குறைந்த ஆற்றல் கொண்ட உதோட்பகுதிகளில் முதலில் இலத்திரன்கள் நிரம்பும்

ஆற்றல் வரிசை:

1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p → 5s → 4d → 5p → 6s → 4f → 5d → 6p

4s 4p 4d 4f

↘ திரிகோண வழியில் படியுங்கள்

பவுலியின் விலக்கல் கோட்பாடு (Pauli Exclusion Principle)

- ஒரு அணுவில் எந்த இரண்டு இலத்திரன்களும் நான்கு குவாண்டம் எண்களிலும் ஒரே மதிப்பை கொண்டிருக்க முடியாது

- ஒரு உதோட்பகுதியில் அதிகபட்சம் 2 இலத்திரன்கள் மட்டுமே இருக்கும் (எதிர் சுழல் கொண்டு)

ஹண்டின் விதி (Hund's Rule)

- ஒரே ஆற்றல் கொண்ட உதோட்பகுதிகளில் (degenerate orbitals) இலத்திரன்கள் முதலில் தனித்தனியாக நிரம்பும்

- தனித்திருக்கும் இலத்திரன்கள் ஒரே சுழல் திசையில் இருக்கும்

3. முக்கிய வரையறைகள்

சொல் தமிழ் விளக்கம்
அணு (Atom) ஒரு தனிமத்தின் இரசாயன பண்புகளை கொண்ட மிகச்சிறிய துகள்
கருவம் (Nucleus) அணுவின் மையத்தில் உள்ள, புரோட்டான்களும் நியூட்ரான்களும் அடங்கிய பகுதி
புரோட்டான் (Proton) நேர்மின் சுமை கொண்ட கருவத்தின் துணிக்கை; ஏற்றம் = +1.602×10⁻¹⁹C
நியூட்ரான் (Neutron) மின்சுமையற்ற கருவத்தின் துணிக்கை
இலத்திரன் (Electron) எதிர்மின் சுமை கொண்ட கருவத்திற்கு வெளியே சுற்றும் துணிக்கை
அணு இலக்கம் (Z) ஒரு அணுவில் உள்ள புரோட்டான்களின் எண்ணிக்கை
நிறை இலக்கம் (A) புரோட்டான்கள் + நியூட்ரான்களின் மொத்த எண்ணிக்கை
ஐசோடோப்புகள் ஒரே Z ஆனால் வேறுபட்ட A கொண்ட அணுக்கள்
ஐசோடோன்கள் ஒரே நியூட்ரான் எண் கொண்ட அணுக்கள்
ஐசோபார்கள் ஒரே A ஆனால் வேறுபட்ட Z கொண்ட அணுக்கள்
உதோட்பகுதி (Orbital) ஒரு இலத்திரன் 90% நேரம் காணப்படும் இடத்தின் நிகழ்தகவு வரைபடம்
ஆற்றல் மட்டம் இலத்திரன்கள் இருக்கும் குறிப்பிட்ட ஆற்றல் நிலை (n=1,2,3...)
ஆற்றல் துணி மட்டம் s, p, d, f உதோட்பகுதிகளின் குழு
இலத்திரன் அமைவு ஒரு அணுவில் இலத்திரன்கள் உதோட்பகுதிகளில் அமைந்திருக்கும் முறை
வேலன்ஸ் இலத்திரன்கள் வெளிப்புற ஆற்றல் மட்டத்தில் உள்ள இலத்திரன்கள்

4. முக்கிய சமன்பாடுகள்

4.1 ஆற்றல்-அலைநீளம் தொடர்பு

$$E = h\nu = \frac{hc}{\lambda}$$

- E = ஒளிக்கதிரின் ஆற்றல் (J)

- h = பிளாங்க் மாறிலி = 6.626 × 10⁻³⁴ J·s

- ν = அலைவெண் (Hz)

- c = ஒளியின் வேகம் = 3.0 × 10⁸ m/s

- λ = அலைநீளம் (m)

4.2 போரின் ஆற்றல் சமன்பாடு (ஹைட்ரஜன் அணு)

$$E_n = -\frac{13.6}{n^2} \text{ eV}$$

அல்லது

$$E_n = -\frac{2.18 \times 10^{-18}}{n^2} \text{ J}$$

- n = முதன்மை குவாண்டம் எண்

- எதிர்குறி = கருவம் இலத்திரனை ஈர்க்கிறது என்பதை குறிக்கும்

4.3 டெ பிரோய் அலை சமன்பாடு

$$\lambda = \frac{h}{mv} = \frac{h}{p}$$

- λ = துகளின் அலைநீளம்

- m = திணிவு

- v = வேகம்

- p = உந்தம்

4.4 ரிட்பர்க் சமன்பாடு

$$\bar{\nu} = \frac{1}{\lambda} = R_H \left(\frac{1}{n_1^2} - \frac{1}{n_2^2}\right)$$

R_H = 1.097 × 10⁷ m⁻¹

4.5 இலத்திரனை பிரிக்க தேவையான ஆற்றல்

$$\Delta E = E_{n_2} - E_{n_1} = -13.6\left(\frac{1}{n_2^2} - \frac{1}{n_1^2}\right) \text{ eV}$$

5. சூத்திரங்கள் மற்றும் விளக்கம்

சூத்திரம் பொருள் பயன்பாடு
A = Z + N நிறை இலக்கம் = புரோட்டான் + நியூட்ரான் அணுவின் கனம் கணக்கிட
max e⁻ = 2n² ஒரு மட்டத்தில் அதிகபட்ச இலத்திரன்கள் இலத்திரன் அமைவு
E = hν ஒளிக்கவந்தத்தின் ஆற்றல் ஒளி அலைவெண் தொடர்பு
λ = c/ν அலைநீளம் மற்றும் அலைவெண் தொடர்பு நிழற்படை கணக்கீடுகள்
mₗ = -l to +l காந்த குவாண்ட எண்ணின் எல்லை உதோட்பகுதி எண்ணிக்கை
2(2l+1) ஒரு துணிமட்டத்தில் அதிகபட்ச இலத்திரன்கள் துணிமட்ட திறன்

6. பரீட்சையில் முக்கியமான பகுதிகள்

அதிகம் கேட்கப்படும் கருத்துகள்:

1. குவாண்டம் எண்கள் - குறிப்பிட்ட இலத்திரனுக்கு எல்லா நான்கு குவாண்டம் எண்களையும் கூறுக

2. இலத்திரன் அமைவு - அணுவிற்கும் அயனிக்கும் இலத்திரன் அமைவு எழுதுக

3. ஒளிர்வு நிழற்படை - ரிட்பர்க் சமன்பாட்டில் கணக்கீடுகள்

4. ஆற்றல் கணக்கீடுகள் - E = hν, E = hc/λ கொண்டு ஆற்றல் கணக்கிடல்

5. ஹண்டின் விதி - உதோட்பகுதி அமைவு வரைபட வினாக்கள்

6. ஐசோடோப்புகள் - ஐசோடோப்பு, ஐசோபார், ஐசோடோன் வேறுபாடுகள்

முக்கிய கண்ணோட்ட வினாக்கள்:

- போரின் மாதிரியின் வரம்புகள் என்ன?

- ஹைசன்பர்க் நிச்சயமின்மைக் கோட்பாட்டின் முக்கியத்துவம் என்ன?

- ஆற்றல் மட்டங்கள் மற்றும் துணிமட்டங்கள் வேறுபாடு என்ன?

7. மாணவர்கள் செய்யும் பொதுவான பிழைகள்

பிழை சரியான முறை

9.2 குவாண்டம் எண்கள் கண்டுபிடிக்கும் முறை

உதாரணம்: 3p⁴ இல் கடைசி இலத்திரன்

- n = 3 (3rd shell)

- l = 1 (p orbital)

- mₗ = ? → p ல் மூன்று உதோட்பகுதிகள்: -1, 0, +1

- ஹண்டின் விதியின்படி: முதல் மூன்று ஒவ்வொன்றில் ஒன்று; நான்காவது -1 ல்

- கடைசி இலத்திரன் (4வது): mₗ = -1

- mₛ = -½ (ஏனெனில் இது இரண்டாவது இலத்திரன் அந்த உதோட்பகுதியில்)

9.3 ஒளிர்வு நிழற்படை கணக்கீடு

வினா: ஹைட்ரஜன் அணுவில் n=3 இல் இருந்து n=1 க்கு இலத்திரன் தாவும்போது வெளிவரும் ஒளியின் அலைநீளம் என்ன?

$$\frac{1}{\lambda} = R_H \left(\frac{1}{1^2} - \frac{1}{3^2}\right) = 1.097 \times 10^7 \left(1 - \frac{1}{9}\right)$$
$$= 1.097 \times 10^7 \times \frac{8}{9} = 9.751 \times 10^6 \text{ m}^{-1}$$
$$\lambda = \frac{1}{9.751 \times 10^6} = 1.026 \times 10^{-7} \text{ m} = 102.6 \text{ nm}$$

இது புறவூதா பகுதியில் உள்ளது (லைமன் தொடர்).

பாடத்தின் நோக்கம்

அணுக்கள் ஒன்றிணைந்து மூலக்கூறுகள் உருவாகின்றன. இந்த இணைப்பு எவ்வாறு உருவாகிறது? என்ன வகையான பிணைப்புகள் உள்ளன? மூலக்கூறுகளின் வடிவம் எவ்வாறு தீர்மானிக்கப்படுகிறது? என்பவற்றை இந்தப் பாடம் விவரிக்கிறது.

முக்கிய கருத்துகள்

- அயனிப் பிணைப்பு: மின்னேற்றங்களுக்கிடையில் மின்னியல் கவர்ச்சி

- சகப்பிணைப்பு: இலத்திரன்களை பகிர்ந்துகொள்வதால் உருவாகும் பிணைப்பு

- VSEPR: மூலக்கூறு வடிவத்தை கணிக்கும் கோட்பாடு

- சக்தி பிணைப்பு வகைகள்: σ, π பிணைப்புகள்

2. முக்கிய கோட்பாடுகள்

2.1 இயனிப் பிணைப்பு (Ionic Bonding)

உருவாக்கம்: ஒரு அணு இலத்திரன் இழக்க (cation), மற்றொரு அணு இலத்திரன் பெற (anion) → எதிர் மின்சுமை அயனிகளுக்கிடையே மின்னியல் கவர்ச்சி

NaCl உருவாக்கம்:

Na → Na⁺ + e⁻  (அயனியாக்க ஆற்றல்: 496 kJ/mol)
Cl + e⁻ → Cl⁻  (இலத்திரன் தொகுப்பு ஆற்றல்: -349 kJ/mol)
Na⁺ + Cl⁻ → NaCl  (கூடு ஆற்றல்: -786 kJ/mol)

அயனிப் பிணைப்பு சேர்மங்களின் பண்புகள்:

- உயர் உருகு மற்றும் கொதி நிலை

- திடவ நிலையில் மின்னழுத்தம் கடத்தாது (அயனிகள் நிலையாக இருக்கும்)

- உருகிய அல்லது கரைசல் நிலையில் மின்னழுத்தம் கடத்தும்

- கடினமானவை, ஆனால் உடையக்கூடியவை

- நீரில் கரையக்கூடியவை (பொதுவாக)

கூடு ஆற்றல் (Lattice Energy):

- அயனிப் பிணைப்பின் வலிமையை அளவிடும்

- Born-Haber சுழற்சி மூலம் கணக்கிடலாம்

- அயனிகளின் சுமை அதிகரிக்க/அயனி ஆரம் குறைய கூடு ஆற்றல் அதிகரிக்கும்

2.2 சகப்பிணைப்பு (Covalent Bonding)

உருவாக்கம்: இரு அணுக்கள் இலத்திரன் ஜோடிகளை பகிர்ந்துகொண்டு உருவாகும் பிணைப்பு

வகைகள்:

பிணைப்பு வகை இலத்திரன் ஜோடிகள் உதாரணம்
ஒற்றைப் பிணைப்பு 1 H-H, C-C
இரட்டைப் பிணைப்பு 2 C=C, O=O
மும்மைப் பிணைப்பு 3 N≡N, C≡C

ஒருதிசைப் பிணைப்பு (Dative/Coordinate Bond):

- இரண்டு இலத்திரன்களும் ஒரு அணுவிடமிருந்தே வரும்

- உதாரணம்: NH₃ + H⁺ → NH₄⁺ (N இன் lone pair → H⁺)

- குறியீடு: →

சகப்பிணைப்பு சேர்மங்களின் பண்புகள்:

- குறைந்த உருகு மற்றும் கொதி நிலை (மூலக்கூறு சேர்மங்கள்)

- பொதுவாக மின்னழுத்தம் கடத்தாது

- ஆர்கானிக் கரைப்பான்களில் கரையக்கூடியவை

2.3 VSEPR கோட்பாடு (Valence Shell Electron Pair Repulsion)

அடிப்படை:

- மத்திய அணுவைச் சுற்றும் இலத்திரன் ஜோடிகள் ஒன்றையொன்று விலக்கும்

- தனி ஜோடிகள் (lone pairs) பிணைப்பு ஜோடிகளை அதிகமாக விலக்கும்

- விலக்கல் வரிசை: lone pair-lone pair > lone pair-bond pair > bond pair-bond pair

VSEPR மூலக்கூறு வடிவங்கள்:

மத்திய அணு பிணைப்பு ஜோடி தனி ஜோடி வடிவம் கோணம் உதாரணம்
2 2 0 நேர்கோடு 180° BeCl₂, CO₂
3 3 0 முக்கோண தளம் 120° BF₃, SO₃
3 2 1 வளைந்த (V வடிவம்) <120° SO₂
4 4 0 நான்முக வடிவம் 109.5° CH₄
4 3 1 முக்கோண பிரமிட் <109.5° NH₃
4 2 2 வளைந்த <109.5° H₂O
5 5 0 முக்கோண இருமுக (Trigonal bipyramidal) 90°,120° PCl₅
6 6 0 எட்டுமுக (Octahedral) 90° SF₆

கோண மாற்றங்கள்:

- NH₃: 107° (தனி ஜோடி → பிணைப்பு ஜோடி விலக்கல் அதிகம்)

- H₂O: 104.5° (இரு தனி ஜோடிகள் → இன்னும் அதிக விலக்கல்)

2.4 பிணைப்பு திட்ட கோட்பாடு (Valence Bond Theory)

σ (sigma) பிணைப்பு:

- இரு அணுக்களின் இடையே நேர் (head-on) மேலோட்டம்

- அணுக்கள் இடையே நேர்க்கோட்டில் இலத்திரன் அடர்த்தி அதிகம்

- ஒற்றைப் பிணைப்பு = 1 σ பிணைப்பு

π (pi) பிணைப்பு:

- p உதோட்பகுதிகளுக்கிடையே பக்க (side-on) மேலோட்டம்

- பிணைப்புக் கோட்டிற்கு மேலும் கீழும் இலத்திரன் அடர்த்தி

- இரட்டைப் பிணைப்பு = 1σ + 1π

- மும்மைப் பிணைப்பு = 1σ + 2π

2.5 கலப்பு உதோட்பகுதிகள் (Hybridization)

sp கலப்பு:

- s + p → 2 sp உதோட்பகுதிகள்

- 180° கோணம் → நேர்கோட்டு வடிவம்

- உதாரணம்: BeCl₂, C₂H₂ (அசிட்டிலீன்), CO₂

sp² கலப்பு:

- s + 2p → 3 sp² உதோட்பகுதிகள்

- 120° கோணம் → முக்கோண தள வடிவம்

- உதாரணம்: BF₃, C₂H₄ (எத்திலீன்), SO₃

sp³ கலப்பு:

- s + 3p → 4 sp³ உதோட்பகுதிகள்

- 109.5° கோணம் → நான்முக வடிவம்

- உதாரணம்: CH₄, NH₃, H₂O, CCl₄

sp³d கலப்பு:

- 5 உதோட்பகுதிகள் → முக்கோண இருமுக

- உதாரணம்: PCl₅

sp³d² கலப்பு:

- 6 உதோட்பகுதிகள் → எட்டுமுக

- உதாரணம்: SF₆

2.6 மூலக்கூறு ஆர்பிட்டால் கோட்பாடு (MO Theory)

அடிப்படைக் கருத்துகள்:

- அணு உதோட்பகுதிகள் இணைந்து மூலக்கூறு உதோட்பகுதிகள் உருவாகும்

- பிணைப்பு MO (σ): உதோட்பகுதிகளின் கட்டமைப்பு கூட்டல் → ஆற்றல் குறையும்

- எதிர்-பிணைப்பு MO (σ*): உதோட்பகுதிகளின் கட்டமைப்பு கழிப்பு → ஆற்றல் அதிகரிக்கும்

பிணைப்பு வரிசை:

$$\text{Bond Order} = \frac{\text{பிணைப்பு MO இலத்திரன்கள்} - \text{எதிர்-பிணைப்பு MO இலத்திரன்கள்}}{2}$$
மூலக்கூறு பிணைப்பு வரிசை நிலைப்புத்தன்மை
H₂ 1 நிலையானது
He₂ 0 இல்லை
O₂ 2 நிலையானது
N₂ 3 மிகவும் நிலையானது

O₂ இன் MO அமைவு:

σ1s² σ*1s² σ2s² σ*2s² σ2p² π2p⁴ π*2p²

→ 2 தனி இலத்திரன்கள் π*2p ல் → O₂ காந்த பண்பு (paramagnetic)

2.7 இடைமூலக்கூறு விசைகள் (Intermolecular Forces)

வான் டெர் வால்ஸ் விசைகள்:

1. பரவல் (Dispersion/London) விசைகள்:

- அனைத்து மூலக்கூறுகளிலும் உள்ளது

- தற்காலிக இருதொடு கவர்ச்சி

- அணு எண் / மூலக்கூற்று திணிவு அதிகர → விசை அதிகரிக்கும்

2. இருதொடு-இருதொடு (Dipole-Dipole) விசைகள்:

- துருவமான மூலக்கூறுகளிடையே

- நேர்மறை நுனி → எதிர்மறை நுனி கவர்ச்சி

3. நீரிய பிணைப்பு (Hydrogen Bond):

- F, O, N உடன் பிணைந்த H மிகவும் துருவமாக்கப்பட்டு இருக்கும்

- அடுத்த மூலக்கூறின் F, O, N உடன் கவர்ச்சி

- விசைகளின் வலிமை வரிசை: H-bond > dipole-dipole > dispersion

- H₂O இன் அசாதாரண உயர் கொதிநிலை (100°C): நீரிய பிணைப்பு காரணம்

கொதிநிலை மற்றும் விசை தொடர்பு:

பொதுவான Group 16: H₂O (100°C) > H₂Te (-2°C) > H₂Se (-41°C) > H₂S (-60°C)
H₂O மிக உயர்ந்த கொதிநிலை → நீரிய பிணைப்பு விளைவு

2.8 உலோகப் பிணைப்பு (Metallic Bonding)

கடல் மாதிரி (Sea of Electrons):

- உலோக அயனிகள் (+) நிலையாக அமைந்திருக்கும்

- விடுதலையான (delocalized) இலத்திரன்கள் ஓடும்

- இலத்திரன்கள் "கடல் போல்" ஓடுவதால் மின்னழுத்தம் கடத்துகின்றன

உலோகப் பண்புகள்:

- மின்னழுத்தம் கடத்தும் (இலத்திரன்கள் சுதந்திரமாக நகரும்)

- வெப்பம் கடத்தும்

- பளபளப்பு (ஒளி எதிரொளிப்பு)

- வளைவுத்தன்மை (malleable) மற்றும் நீட்சித்தன்மை (ductile)

- உயர் உருகு நிலை

3. முக்கிய வரையறைகள்

சொல் தமிழ் விளக்கம்
அயனிப் பிணைப்பு எதிர்மின் சுமை அயனிகளுக்கிடையே மின்னியல் கவர்ச்சியால் உருவான பிணைப்பு
சகப்பிணைப்பு இரண்டு அணுக்களும் இலத்திரன் ஜோடியை பகிர்ந்துகொண்டு உருவாக்கும் பிணைப்பு
ஒருதிசைப் பிணைப்பு இரு இலத்திரன்களும் ஒரே அணுவிடமிருந்து வரும் சகப்பிணைப்பு
ஒற்றைப் பிணைப்பு ஒரு இலத்திரன் ஜோடி மட்டுமே பகிரப்படும் (σ பிணைப்பு)
இரட்டைப் பிணைப்பு இரண்டு இலத்திரன் ஜோடிகள் பகிரப்படும் (1σ + 1π)
மும்மைப் பிணைப்பு மூன்று இலத்திரன் ஜோடிகள் பகிரப்படும் (1σ + 2π)
VSEPR வாலன்ஸ் ஷெல் இலத்திரன் ஜோடி விலக்கல் கோட்பாடு
கலப்பு உதோட்பகுதி இரண்டு அல்லது அதிக உதோட்பகுதிகள் கலந்து உருவான சமஆற்றல் உதோட்பகுதிகள்
இலத்திரன் ஈர்ப்புத்திறன் (Electronegativity) ஒரு அணு பகிர்ந்துகொள்ளப்பட்ட இலத்திரன்களை இழுக்கும் திறன்
துருவ பிணைப்பு இலத்திரன்கள் சமனாக பகிரப்படாத சகப்பிணைப்பு
நீரிய பிணைப்பு H…F/O/N இடையே வலுவான இடைமூலக்கூறு கவர்ச்சி

4. முக்கிய சமன்பாடுகள்

4.1 பிணைப்பு தொகுப்பு (Born-Haber cycle)

NaCl உருவாக்கத்திற்கு:

$$\Delta H_f = \Delta H_{sub}(Na) + \Delta H_{IE1}(Na) + \frac{1}{2}\Delta H_{dissociation}(Cl_2) + \Delta H_{EA}(Cl) + U(NaCl)$$

4.2 பிணைப்பு வரிசை (Bond Order - MO Theory)

$$\text{B.O.} = \frac{(\text{பிணைப்பு MO } e^-) - (\text{எதிர்-பிணைப்பு MO } e^-)}{2}$$

4.3 மின் இருதொடு விகிதம் (Dipole Moment)

$$\mu = q \times d$$

- q = ஏற்றத்தின் அளவு

- d = பிணைப்பு நீளம்

5. சூத்திரங்கள் மற்றும் விளக்கம்

இலட்சிய வாயு சமன்பாடு:

$$PV = nRT$$

- P = அழுத்தம் (Pa அல்லது atm)

- V = கன அளவு (m³ அல்லது L)

- n = மோல்கள்

- R = வாயு மாறிலி = 8.314 J/mol·K = 0.0821 L·atm/mol·K

- T = வெப்பநிலை (K)

STP (Standard Temperature and Pressure): 0°C, 1 atm → 1 மோல் வாயு = 22.4 L

RTP (Room Temperature and Pressure): 25°C, 1 atm → 1 மோல் வாயு = 24.0 L

Boyle's Law: P₁V₁ = P₂V₂ (T மாறாது)

Charles's Law: V₁/T₁ = V₂/T₂ (P மாறாது)

Gay-Lussac's Law: P₁/T₁ = P₂/T₂ (V மாறாது)

Combined Law: P₁V₁/T₁ = P₂V₂/T₂

2.7 தீர்வுகள் (Solutions)

செறிவு வகைகள்:

1. மோலாரிட்டி (M):

$$M = \frac{\text{மோல்கள்}}{\text{லிட்டரில் கன அளவு}} = \frac{n}{V(L)}$$

2. மோலாலிட்டி (m):

$$m = \frac{\text{மோல்கள்}}{\text{கிலோகிராமில் கரைப்பான்}}$$

3. மோல் பின்னம் (Mole Fraction):

$$\chi_A = \frac{n_A}{n_A + n_B}$$

4. திணிவு சதவீதம் (Mass %):

$$\text{mass\%} = \frac{m_{solute}}{m_{solution}} \times 100$$

திட்டப்படுத்தல் (Dilution):

$$M_1V_1 = M_2V_2$$

2.8 அணுத்திணிவு கணக்கீடு

ஐசோடோப்பு திணிவிலிருந்து சராசரி அணுத்திணிவு:

$$\bar{A} = \sum (A_i \times \text{abundance}_i)$$

உதாரணம் - குளோரீன் (Cl):

- Cl-35: 75% (0.75), அணுத்திணிவு = 34.969

- Cl-37: 25% (0.25), அணுத்திணிவு = 36.966

$$\bar{A} = (0.75 \times 34.969) + (0.25 \times 36.966) = 35.45$$

2.9 கூடு ஆற்றல் (Lattice Energy) - Born-Haber Cycle

Hess's Law: இரசாயன வினையின் ஆற்றல் மாற்றம் வழியை சார்ந்திருக்காது.

Born-Haber Cycle for NaCl:

Na(s) + ½Cl₂(g) → NaCl(s)    ΔHf
↓ ΔHsub(Na)        ↑ U (Lattice Energy)
Na(g) + ½Cl₂(g)
↓ ΔH_IE1(Na)   ↓ ½ΔH_diss(Cl₂)
Na⁺(g) + Cl(g)
          ↓ ΔH_EA(Cl)
Na⁺(g) + Cl⁻(g) ──────────────────
$$\Delta H_f = \Delta H_{sub} + \Delta H_{IE1} + \frac{1}{2}\Delta H_{diss} + \Delta H_{EA} + U$$

அறிமுகம்

1. பாட அறிமுகம்

இரசாயன வினைகளில் ஆற்றல் மாற்றங்களை ஆய்வு செய்வது வெப்பவேதியியல். "வினையில் வெப்பம் வெளியேறுமா? உறிஞ்சப்படுமா?" என்பதே அடிப்படை வினா.

2. முக்கிய கோட்பாடுகள்

2.1 வெப்பவேதியியல் அடிப்படைகள்

ஒலி அமைப்பு (System): ஆய்வுக்கு உட்பட்ட பகுதி

சூழல் (Surroundings): மீதமுள்ள எல்லாமே

வெப்ப இயக்கவியல் 1ஆம் விதி (First Law):

$$\Delta U = q + w$$

- ΔU = உட்சக்தி மாற்றம்

- q = வெப்பம் (நேர் = உறிஞ்சல்; எதிர் = வெளியேற்றம்)

- w = வேலை (−PΔV)

மாறா அழுத்தத்தில்:

$$\Delta H = q_p$$

H = Enthalpy (ஆற்றல் உள்ளடக்கம்)

2.2 ΔH வகைகள்

வகை குறியீடு வரையறை
உருவாக்க ஆற்றல் ΔHf° 1 mol சேர்மம் தூய தனிமங்களிலிருந்து உருவாக்கும் ஆற்றல்
எரிப்பு ஆற்றல் ΔHc° 1 mol பொருள் முழுமையாக எரியும்போது வெளியேறும் ஆற்றல்
தணிப்பு ஆற்றல் ΔHneutralization அமிலம் + காரம் → 1 mol H₂O உருவாகும் ஆற்றல்
கரைவு ஆற்றல் ΔHsolution 1 mol கரைதிரவம் கரையும்போது ஆற்றல்
அயனியாக்க ஆற்றல் IE வாயு நிலை அணுவிலிருந்து 1 mol e⁻ நீக்கும் ஆற்றல்

2.3 ஹெஸ்ஸின் விதி (Hess's Law)

கோட்பாடு: ஒரு வினையின் ΔH மாற்றம் வழியை சார்ந்திருக்காது.

பயன்பாடு: நேரடியாக அளவிட முடியாத வினைகளுக்கு ΔH கணக்கிட

சூத்திரம்:

$$\Delta H_{reaction} = \sum \Delta H_f°(\text{products}) - \sum \Delta H_f°(\text{reactants})$$

உதாரணம்:

$$C + O_2 \rightarrow CO_2 \quad \Delta H = -393.5 \text{ kJ/mol}$$
$$C + \frac{1}{2}O_2 \rightarrow CO \quad \Delta H = -110.5 \text{ kJ/mol}$$
$$CO + \frac{1}{2}O_2 \rightarrow CO_2 \quad \Delta H = ?$$

தீர்வு: (1) - (2) → ΔH = -393.5 - (-110.5) = -283 kJ/mol

2.4 பிணைப்பு ஆற்றல் கொண்டு ΔH கணிப்பு

$$\Delta H = \sum \text{(உடைக்கப்படும் பிணைப்புகளின் ஆற்றல்)} - \sum \text{(உருவாகும் பிணைப்புகளின் ஆற்றல்)}$$

முக்கிய பிணைப்பு ஆற்றல்கள்:

பிணைப்பு ஆற்றல் (kJ/mol)

Cl-Cl 243
H-Cl 432
C-H 412
C=C 612
C≡C 837
O=O 498
O-H 463

2.5 Born-Haber சுழற்சி

NaCl க்கான Born-Haber Cycle:

$$\Delta H_f = \Delta H_{sub}(Na) + IE_1(Na) + \frac{1}{2}D(Cl_2) + EA(Cl) + U$$

- ΔH_sub = ஊறுகாய் ஆற்றல் (Sublimation)

- IE₁ = முதல் அயனியாக்க ஆற்றல்

- D = விலகல் ஆற்றல் (Dissociation)

- EA = இலத்திரன் தொகுப்பு ஆற்றல்

- U = கூடு ஆற்றல் (Lattice Energy)

2.6 வெப்பவேதியியல் 2ஆம் விதி - எண்ட்ரோபி (Entropy)

எண்ட்ரோபி (S): ஒரு மண்டலத்தின் ஒழுங்கின்மை அல்லது சீரற்ற தன்மை

எண்ட்ரோபி மாற்றங்கள்:

- திட → திரவம் → வாயு: S அதிகரிக்கும்

- வாயு மூலக்கூறுகள் அதிகரிக்க: S அதிகரிக்கும்

- கலத்தல் (mixing): S அதிகரிக்கும்

2ஆம் விதி: தன்னிச்சையான வினைகளில் பிரபஞ்சத்தின் மொத்த S அதிகரிக்கும்

$$\Delta S_{universe} = \Delta S_{system} + \Delta S_{surroundings} > 0$$

2.7 கிப்ஸ் சுதந்திர ஆற்றல் (Gibbs Free Energy)

$$\Delta G = \Delta H - T\Delta S$$
ΔH ΔS ΔG தன்னிச்சை?
− + − எப்போதும் எப்போதும் தன்னிச்சை
+ − + எப்போதும் ஒருபோதும் தன்னிச்சை இல்லை
− − T சார்ந்தது குறைந்த T ல் தன்னிச்சை
+ + T சார்ந்தது அதிக T ல் தன்னிச்சை

சமநிலையில்: ΔG = 0 → T = ΔH/ΔS

3. முக்கிய வரையறைகள்

சொல் தமிழ் விளக்கம்
என்தால்பி (H) மாறா அழுத்தத்தில் ஒரு மண்டலத்தின் மொத்த வெப்ப உள்ளடக்கம்
தன்னிடைவெப்ப வினை வெப்பம் வெளியேறும் வினை (ΔH < 0)
வெப்பமிடைவெப்ப வினை வெப்பம் உறிஞ்சப்படும் வினை (ΔH > 0)
எண்ட்ரோபி (S) ஒழுங்கின்மையின் அளவு
கிப்ஸ் ஆற்றல் (G) ΔG = ΔH - TΔS; வினையின் தன்னிச்சைத்தன்மை அறிய
Hess's Law ΔH வழியை சார்ந்திருக்காது
கலோரிமீட்டர் வெப்ப மாற்றம் அளவிடும் கருவி

4. கணக்கீட்டு உதாரணங்கள்

கணக்கு 1: Hess's Law

ΔHf° (CO) கண்டுபிடிக்க:

- C + O₂ → CO₂, ΔH₁ = -393.5 kJ

- CO + ½O₂ → CO₂, ΔH₂ = -283 kJ

தீர்வு:

வினை (1) - வினை (2):

C + O₂ - CO - ½O₂ → 0

C + ½O₂ → CO, ΔH = -393.5 - (-283) = -110.5 kJ

கணக்கு 2: பிணைப்பு ஆற்றல் கொண்டு ΔH

H₂ + Cl₂ → 2HCl, ΔH = ?

உடைக்கப்படும்: H-H (436) + Cl-Cl (243) = 679 kJ

உருவாகும்: 2 × H-Cl (432) = 864 kJ

ΔH = 679 - 864 = -185 kJ

5. சுருக்க குறிப்புகள்

🔑 ΔH > 0 → வெப்பம் உறிஞ்சல் (endothermic)
🔑 Hess's Law: ΔH_rxn = ΣΔHf°(products) - ΣΔHf°(reactants)
🔑 ΔG = ΔH - TΔS | ΔG < 0 → தன்னிச்சை
🔑 S: gas > liquid > solid; mixing increases S
🔑 பிணைப்பு ஆற்றல்: உடைக்க = + (உறிஞ்சல்), உருவாக்க = - (வெளியேற்றம்)

பாடம் 5: வினைவேகவியல்

Chemical Kinetics
சிரமநிலை: ⭐⭐⭐⭐ முக்கியத்துவம்: ⭐⭐⭐⭐⭐ நேரம்: ** 12-15 மணித்தியாலங்கள்

அறிமுகம்

1. பாட அறிமுகம்

வினை எவ்வளவு வேகமாக நடக்கிறது? என்ன காரணிகள் வேகத்தை மாற்றுகின்றன? இதுவே வினைவேகவியல்.

2. முக்கிய கோட்பாடுகள்

2.1 வினை வேகம் (Rate of Reaction)

$$\text{Rate} = \frac{\Delta[\text{product}]}{\Delta t} = -\frac{\Delta[\text{reactant}]}{\Delta t}$$

வேகத்தை பாதிக்கும் காரணிகள்:

1. செறிவு - அதிக செறிவு → அதிக மோதல்கள் → வேகம் அதிகரிக்கும்

2. வெப்பநிலை - அதிக வெப்பநிலை → மோதல் ஆற்றல் அதிகரிக்கும்

3. கலப்பியம் (Catalyst) - செயல்படுத்தல் ஆற்றல் குறைக்கும்

4. மேற்பரப்பு பரப்பளவு - துகள் அளவு குறைய மேற்பரப்பு அதிகரிக்கும்

5. அழுத்தம் (வாயு வினைகளில்) - செறிவிற்கு சமம்

2.2 வேக விதி (Rate Law)

$$\text{Rate} = k[A]^m[B]^n$$

- k = வேக மாறிலி

- m = A ஐப் பொறுத்த வரிசை

- n = B ஐப் பொறுத்த வரிசை

- (m + n) = மொத்த வரிசை

வரிசை வகைகள்:

வரிசை வேக விதி ஒருங்கிணைந்த விதி t₁/₂
0 Rate = k [A] = [A]₀ - kt [A]₀/2k
1 Rate = k[A] [A] = [A]₀e^(-kt) ln2/k = 0.693/k
2 Rate = k[A]² 1/[A] = 1/[A]₀ + kt 1/(k[A]₀)

2.3 செயல்படுத்தல் ஆற்றல் (Activation Energy - Ea)

- வினை நடக்க தேவையான குறைந்தபட்ச ஆற்றல்

- கலப்பியம் Ea ஐ குறைக்கும்

ஆரீனியஸ் சமன்பாடு:

$$k = Ae^{-E_a/RT}$$

அல்லது:

$$\ln k = \ln A - \frac{E_a}{RT}$$

இரண்டு வெப்பநிலைகளில்:

$$\ln\frac{k_2}{k_1} = \frac{E_a}{R}\left(\frac{1}{T_1} - \frac{1}{T_2}\right)$$

2.4 மோதல் கோட்பாடு (Collision Theory)

வினை நடக்க இரண்டு நிபந்தனைகள்:

1. மோதல் சரியான திசையில் இருக்க வேண்டும் (Orientation)

2. மோதல் ஆற்றல் ≥ Ea

2.5 வினை வழிமுறை (Reaction Mechanism)

- சிக்கலான வினை படிப்படியாக நடக்கும்

- வேகத்தை தீர்மானிக்கும் படி = மிக மெதுவான படி (Rate-determining step)

- இடைப்பட்ட சேர்மங்கள் (Intermediates) = உருவாகி உடனே தீர்க்கப்படும்

- கலப்பியம் = வினையில் பங்கேற்று இறுதியில் மீண்டும் கிடைக்கும்

3. வரையறைகள் அட்டவணை

சொல் விளக்கம்
வேக மாறிலி (k) செறிவு = 1 ஆக இருக்கும்போது வினை வேகம்
அரை ஆயுள் (t₁/₂) செறிவு பாதியாக குறைய ஆகும் நேரம்
செயல்படுத்தல் ஆற்றல் வினை தடையை கடக்க தேவையான குறைந்தபட்ச ஆற்றல்
கலப்பியம் வேகத்தை அதிகரித்து, வினையில் இறுதியில் மாறாது தங்கும் பொருள்
இடைப்பட்ட சேர்மம் வினை நடுவில் உருவாகி உடனே வினையாடும் பொருள்

4. சுருக்க குறிப்புகள்

🔑 Rate = k[A]^m[B]^n (Rate law)
🔑 First order: t₁/₂ = 0.693/k (செறிவை சாராது)
🔑 Arrhenius: k = Ae^(-Ea/RT)
🔑 Catalyst: Ea குறைக்கும், K மாற்றாது
🔑 Temperature ↑ → k ↑ (பொதுவாக 10°C அதிகரிக்க rate 2-3 மடங்காகும்)
🔑 Rate-determining step = slowest step

பாடம் 6: சமநிலை

Chemical Equilibrium
சிரமநிலை: ⭐⭐⭐⭐ முக்கியத்துவம்: ⭐⭐⭐⭐⭐ நேரம்: ** 20-25 மணித்தியாலங்கள்

அறிமுகம்

1. பாட அறிமுகம்

தலைகீழ் வினைகளில் முன்வினை வேகம் = பின்வினை வேகம் ஆகும்போது இரசாயன சமநிலை நிகழும். இது மாறா நிலை, ஆனால் வினைகள் தொடர்ந்து நடக்கின்றன (இயக்க சமநிலை).

2. முக்கிய கோட்பாடுகள்

2.1 சமநிலை மாறிலி (Equilibrium Constant)

aA + bB ⇌ cC + dD என்ற வினைக்கு:

$$K_c = \frac{[C]^c[D]^d}{[A]^a[B]^b}$$

வாயு நிலைக்கு (Kp):

$$K_p = \frac{P_C^c \cdot P_D^d}{P_A^a \cdot P_B^b}$$

Kc மற்றும் Kp தொடர்பு:

$$K_p = K_c(RT)^{\Delta n}$$

Δn = வாயு நிலையில் (products மோல் - reactants மோல்)

2.2 லீ சாட்லியரின் கோட்பாடு (Le Chatelier's Principle)

"ஒரு சமநிலை மண்டலத்தில் மாற்றம் உண்டாக்கினால், அம்மாற்றத்தை குறைக்கும் திசையில் சமநிலை நகரும்"

மாற்றங்களும் பதில்களும்:

மாற்றம் சமநிலை நகரும் திசை
அதிக reactant சேர் முன்வினை திசை (→)
அதிக product சேர் பின்வினை திசை (←)
reactant அகற்று பின்வினை திசை (←)
வெப்பநிலை அதிகரி (exothermic) பின்வினை திசை (←)
வெப்பநிலை குறை (exothermic) முன்வினை திசை (→)
அழுத்தம் அதிகரி குறைவான மோல் திசை
அழுத்தம் குறை அதிக மோல் திசை

கலப்பியம்: சமநிலை நிலையை மாற்றாது (Kc மாறாது); ஆனால் வேகமாக சமநிலை அடைய உதவும்

2.3 அமிலம்-காரம் சமநிலை

Brønsted-Lowry கோட்பாடு:

- அமிலம் = H⁺ இழக்கும் பொருள் (proton donor)

- காரம் = H⁺ ஏற்கும் பொருள் (proton acceptor)

ஒருங்கிணைந்த அமிலம்-காரம் ஜோடி:

CH₃COOH ⇌ CH₃COO⁻ + H⁺
(அமிலம்)     (ஒருங்கிணைந்த காரம்)

Kw = [H⁺][OH⁻] = 1.0 × 10⁻¹⁴ (25°C)

pH = -log[H⁺]

pOH = -log[OH⁻]

pH + pOH = 14 (25°C)

2.4 பலவீனமான அமிலம் சமநிலை

HA ⇌ H⁺ + A⁻

$$K_a = \frac{[H^+][A^-]}{[HA]}$$
$$pH = \frac{1}{2}(pK_a - \log C_a)$$

பலவீன காரம்:

$$K_b = \frac{[BH^+][OH^-]}{[B]}$$

Ka × Kb = Kw

2.5 பஃபர் கரைசல்கள் (Buffer Solutions)

வரையறை: pH ஐ மாறாமல் பேணும் கரைசல்

வகைகள்:

1. அமில பஃபர்: பலவீன அமிலம் + அதன் உப்பு (CH₃COOH + CH₃COONa)

2. கார பஃபர்: பலவீன காரம் + அதன் உப்பு (NH₃ + NH₄Cl)

Henderson-Hasselbalch சமன்பாடு:

$$pH = pK_a + \log\frac{[\text{A}^-]}{[\text{HA}]}$$

2.6 கரைவுத் தயாரிப்பு (Solubility Product - Ksp)

AgCl ⇌ Ag⁺ + Cl⁻

$$K_{sp} = [Ag^+][Cl^-]$$

பொது அயன் விளைவு (Common Ion Effect):

பொது அயன் சேர்க்கும்போது கரைவுத்திறன் குறையும்

உதாரணம்: NaCl கரைசலில் AgCl இன் கரைவுத்திறன் குறையும் (Cl⁻ பொது அயன்)

3. வரையறைகள் அட்டவணை

சொல் விளக்கம்
சமநிலை முன்வினை வேகம் = பின்வினை வேகம்
Kc மோல் செறிவுகளில் சமநிலை மாறிலி
Kp பகுதி அழுத்தங்களில் சமநிலை மாறிலி
pH -log[H⁺]; அமிலத்தன்மை அளவீடு
பஃபர் pH மாறாமல் பேணும் கரைசல்
Ksp கரைவுத் தயாரிப்பு மாறிலி
Ka அமிலத்தன்மை மாறிலி (பலவீன அமிலத்திற்கு)
Kb காரத்தன்மை மாறிலி

4. கணக்கீட்டு உதாரணங்கள்

கணக்கு 1: pH கணிப்பு

0.1 M CH₃COOH, Ka = 1.8 × 10⁻⁵. pH = ?

$$[H^+] = \sqrt{K_a \times C} = \sqrt{1.8 \times 10^{-5} \times 0.1} = 1.34 \times 10^{-3}$$
$$pH = -\log(1.34 \times 10^{-3}) = 2.87$$

கணக்கு 2: Kp கணிப்பு

N₂ + 3H₂ ⇌ 2NH₃, Kc = 4.0 × 10⁻⁵ at 500K

Δn = 2 - (1+3) = -2

$$K_p = K_c(RT)^{\Delta n} = 4.0 \times 10^{-5} \times (0.0821 \times 500)^{-2}$$
$$K_p = 4.0 \times 10^{-5} \times (41.05)^{-2} = 2.37 \times 10^{-8}$$

5. சுருக்க குறிப்புகள்

🔑 Kc = [products]^n / [reactants]^m (மாறா T ல்)
🔑 Kp = Kc(RT)^Δn
🔑 Le Chatelier: disturbance → equilibrium shifts to reduce it
🔑 Catalyst: Kc மாறாது, சமநிலை வேகமாக அடையும்
🔑 pH + pOH = 14 (at 25°C)
🔑 pH = pKa → [acid] = [salt] (buffer at best pH)
🔑 Common ion effect: solubility decreases
🔑 Ka × Kb = Kw = 10⁻¹⁴

பாடம் 7: மின்வேதியியல்

Electrochemistry
சிரமநிலை: ⭐⭐⭐⭐ முக்கியத்துவம்: ⭐⭐⭐⭐⭐ நேரம்: ** 15-18 மணித்தியாலங்கள்

அறிமுகம்

2. முக்கிய கோட்பாடுகள்

2.1 ஒட்சியேற்றம்-ஒடுக்கம் (Redox)

ஒட்சியேற்றம்: இலத்திரன்கள் இழத்தல் (OIL)

ஒடுக்கம்: இலத்திரன்கள் பெறுதல் (RIG)

OIL RIG - நினைவூட்டல்

ஒட்சியேற்றி: மற்றதை ஒட்சியேற்றும்; தானே ஒடுக்கப்படும்

ஒடுக்கி: மற்றதை ஒடுக்கும்; தானே ஒட்சியேற்றப்படும்

2.2 மின்னரு கலம் (Galvanic/Voltaic Cell)

கூறுகள்:

- அனோடு (Anode): ஒட்சியேற்றம் நடக்கும் எதிர்மின் முனை (−)

- கேதோடு (Cathode): ஒடுக்கம் நடக்கும் நேர்மின் முனை (+)

- உப்பு பாலம் (Salt Bridge): மின் சுற்றை நிரைத்திருக்க வைக்கும்

- வெளி சுற்று: இலத்திரன்கள் அனோட்டிலிருந்து கேதோட்டுக்கு

டேனியல் கலம் (Daniel Cell):

Zn | ZnSO₄ || CuSO₄ | Cu
அனோடு           கேதோடு
Zn → Zn²⁺ + 2e⁻  |  Cu²⁺ + 2e⁻ → Cu

EMF = E°(cathode) - E°(anode)

2.3 நிலையான மின்னழுத்தம் (Standard Electrode Potential - E°)

நிலையான ஹைட்ரஜன் மின்முனை (SHE): E° = 0.00 V (உசாத்துணை)

நெர்ண்ட் சமன்பாடு:

$$E = E° - \frac{RT}{nF}\ln Q = E° - \frac{0.0592}{n}\log Q \text{ (at 25°C)}$$

முக்கிய E° மதிப்புகள்:

அரை வினை E° (V)
F₂ + 2e⁻ → 2F⁻ +2.87
MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O +1.51
Cl₂ + 2e⁻ → 2Cl⁻ +1.36
O₂ + 4H⁺ + 4e⁻ → 2H₂O +1.23
Cu²⁺ + 2e⁻ → Cu +0.34
2H⁺ + 2e⁻ → H₂ 0.00
Fe²⁺ + 2e⁻ → Fe -0.44
Zn²⁺ + 2e⁻ → Zn -0.76
Li⁺ + e⁻ → Li -3.05

E°cell = E°cathode − E°anode > 0 → தன்னிச்சை வினை

ΔG° மற்றும் E° தொடர்பு:

$$\Delta G° = -nFE°_{cell}$$
$$\ln K = \frac{nFE°}{RT} = \frac{nE°}{0.0257}$$

2.4 மின்னாற்பகுப்பு (Electrolysis)

இதில் மின்னாற்றல் இரசாயன வினையை தூண்டும்

- அனோடு: ஒட்சியேற்றம் (+)

- கேதோடு: ஒடுக்கம் (−) ← மின்னாரு கலத்திற்கு எதிர்

ஃபாரடே விதிகள்:

1. வினையடையும் பொருளின் திணிவு, சுற்றப்பட்ட மின்னழுத்தத்திற்கு நேர் விகிதம்

2. ஒரே மின்னழுத்தத்தில் வேறு வேறு தனிமங்கள் தங்கள் சமான திணிவில் (equivalent weight) வினையடையும்

$$m = \frac{ItM}{nF}$$

- m = வைக்கப்பட்ட திணிவு (g)

- I = மின்னழுத்தம் (A)

- t = நேரம் (s)

- M = மோலார் திணிவு

- n = மாற்றப்பட்ட இலத்திரன்கள்

- F = ஃபாரடே மாறிலி = 96485 C/mol

- O₃ (ozonolysis): C=C பிரிக்கும்

வினைகள்: ஆல்கீன் போலவே, ஆனால் இரண்டு மடங்கு சேர்க்கும்

அரோமேட்டிக் சேர்மங்கள் - பென்சீன் (C₆H₆)

- 6 C ஒரு தளத்தில் ஷட்கோண அமைப்பு

பண்புகள்:

- 4n+2 π electrons (Hückel விதி, n=1 → 6 electrons) → aromatic

- கூட்டல் வினைகளை விட மாற்றீட்டு வினைகள் செய்யும்

வினைகள்:

1. நைட்ரேற்றம் (Nitration):

C₆H₆ + HNO₃ → C₆H₅NO₂ + H₂O (conc. H₂SO₄, 55°C)

2. ஹேலோஜனேற்றம்:

C₆H₆ + Cl₂ → C₆H₅Cl + HCl (FeCl₃ catalyst)

3. சல்ஃபோனேற்றம் (Sulfonation):

C₆H₆ + RCl → C₆H₅R + HCl (AlCl₃ catalyst)

5. ஃபிரீடெல் கிரஃப்ட் அசைலேற்றம்:

C₆H₆ + RCOCl → C₆H₅COR + HCl (AlCl₃ catalyst)

பென்சீன் EAS வழிமுறை:

படி 1: Electrophile (E⁺) உருவாக்கல்
படி 2: E⁺ + C₆H₆ → Sigma complex (arenium ion)
படி 3: H⁺ இழத்தல் → C₆H₅E (product)

2. முக்கிய கோட்பாடுகள்

2.1 ஹேலோஆல்கேன்கள் (Haloalkanes / Alkyl Halides)

பொது சூத்திரம்: R-X (X = F, Cl, Br, I)

வகைகள்:

- Primary (1°): R-CH₂-X (X உடன் ஒரு C)

- Secondary (2°): R₂CH-X (X உடன் இரண்டு C)

- Tertiary (3°): R₃C-X (X உடன் மூன்று C)

முக்கிய வினைகள்:

1. நியூக்ளியோஃபிலிக் மாற்றீட்டு வினை (SN):

R-X + Nu⁻ → R-Nu + X⁻

SN2 SN1
Primary halide Tertiary halide
Backside attack Carbocation intermediate
Inversion (Walden inversion) Racemization
One step Two step

2. நீக்கல் வினை (Elimination - E2/E1):

R-CH₂-CHX-R → R-CH=CH-R + HX (strong base)

Zaitsev விதி: அதிக மாற்றீட்டு alkene முக்கிய பொருளாகும்

3. Grignard வினைப்பொருள் (RMgX):

R-X + Mg → R-MgX (dry ether)

→ aldehydes/ketones உடன் வினையாடி alcohols தரும்

2.2 ஆல்கஹால்கள் (Alcohols) - R-OH

வகைகள்: Primary (1°), Secondary (2°), Tertiary (3°)

முக்கிய பண்புகள்:

- நீரிய பிணைப்பு → உயர் கொதிநிலை

- நீரில் கரையக்கூடியது (சிறிய alcohols)

முக்கிய வினைகள்:

1. உலோகங்களுடன்: 2ROH + 2Na → 2RONa + H₂↑

(1° > 2° > 3° வினைத்திறன் வரிசை)

2. ஹேலோஜன் அமிலங்களுடன்:

ROH + HX → RX + H₂O (3° > 2° > 1° வேகம்)

ROH + SOCl₂ → RCl + SO₂ + HCl

3. ஒட்சியேற்றம்:

   1° Alcohol → (K₂Cr₂O₇/H₂SO₄) → Aldehyde → Carboxylic acid
   2° Alcohol → (K₂Cr₂O₇/H₂SO₄) → Ketone
   3° Alcohol → எளிதில் ஒட்சியேற்றமாகாது

4. நீர் நீக்கம் (Dehydration):

   ROH → (conc. H₂SO₄, 170°C) → Alkene + H₂O
   ROH → (conc. H₂SO₄, 140°C) → Ether (R-O-R)

5. மிளிரடித்தல் (Esterification):

R-COOH + R'OH ⇌ R-COO-R' + H₂O (conc. H₂SO₄)

ஒட்சியேற்றம் சோதனை:

- K₂Cr₂O₇/H₂SO₄: ஆரஞ்சு → பச்சை (primary/secondary alcohol உடன்)

- Fehling's: நீலம் → சிவப்பு (aldehyde உடன்)

- Tollens': Ag mirror (aldehyde உடன்)

2.3 எதர்கள் (Ethers) - R-O-R'

பண்புகள்:

- குறைந்த கொதிநிலை (H-bond இல்லை)

- நல்ல கரைப்பான்

- பொதுவாக வினைத்திறன் குறைவு

தயாரிப்பு:

2ROH → R-O-R + H₂O (conc. H₂SO₄, 140°C)

Williamson synthesis: RO⁻Na⁺ + R'X → R-O-R' + NaX

2.4 கார்போனைல் சேர்மங்கள்

ஆல்டிஹைடுகள் (Aldehydes) - RCHO:

கீட்டோன்கள் (Ketones) - RCOR':

கார்போனைல் வினைகள்:

1. நியூக்ளியோஃபிலிக் சேர்த்தல் (Nucleophilic Addition):

   RCHO + HCN → RCH(OH)CN (cyanohydrin)
   RCHO + NaBH₄/LiAlH₄ → RCH₂OH (reduction)
   RCHO + Grignard (R'MgX) → RR'CHOH (secondary alcohol)

2. ஆல்டோல் சேர்த்தல்:

2CH₃CHO → CH₃CH(OH)CH₂CHO (NaOH)

3. ஒட்சியேற்றம்:

- Aldehydes: Tollens', Fehling's சோதனையில் நேர்

- Ketones: பொதுவாக ஒட்சியேற்றமாகாது (Tollens'/Fehling's-ல் எதிர்)

4. iodoform சோதனை:

CH₃COR + I₂/NaOH → CHI₃↓ (மஞ்சள் அவக்கேதம்) + RCOONa

- CH₃OH, CH₃CHO, CH₃COR, CH₃CH(OH)R: எல்லாம் நேர்

2.5 கார்பாக்சிலிக் அமிலங்கள் (Carboxylic Acids) - RCOOH

அமிலத்தன்மை: Ka அளவீடு

முக்கிய வினைகள்:

1. நடுநிலைப்படுத்தல்: RCOOH + NaOH → RCOONa + H₂O

2. ஈஸ்டர் உருவாக்கல்: RCOOH + R'OH ⇌ RCOOR' + H₂O

3. அமில குளோரைடு: RCOOH + SOCl₂ → RCOCl + SO₂ + HCl

4. ஒடுக்கம்: RCOOH + LiAlH₄ → RCH₂OH

Acyl Chloride (RCOCl) வினைகள்:

- RCOCl + H₂O → RCOOH + HCl

- RCOCl + ROH → RCOOR' + HCl (fast esterification)

- RCOCl + NH₃ → RCONH₂ + HCl (amide)

3. சுருக்க குறிப்புகள்

🔑 SN2: primary, one step, inversion | SN1: tertiary, two step, racemization
🔑 Zaitsev: major product = more substituted alkene
🔑 Grignard: RMgX → nucleophile for C=O

🔑 Esterification: acid + alcohol → ester + water (H₂SO₄ cat.)

அமில-காரம் தகராறு (Acid-Base Titration)

சோடியம் ஹைட்ராக்சைடு vs ஹைட்ரோகுளோரிக் அமிலம்:

குறிகாட்டி (Indicators):

குறிகாட்டி pH வரம்பு அமிலத்தில் காரத்தில்

குறிகாட்டி தேர்வு:

- Strong acid + Strong base → எந்த குறிகாட்டியும்

- Weak acid + Strong base → Phenolphthalein (pH > 7)

- Strong acid + Weak base → Methyl orange (pH < 7)

நிலைமான்ப்படுத்தல்: Primary standard - Sodium carbonate (Na₂CO₃)

ஒட்சியேற்றம்-ஒடுக்கம் தகராறு (Redox Titration)

MnO₄⁻ + 8H⁺ + 5e⁻ → Mn²⁺ + 4H₂O (E° = +1.51 V)

Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O

- குறிகாட்டி: N-phenylanthranilic acid அல்லது diphenylamine

I₂ + 2Na₂S₂O₃ → 2NaI + Na₂S₄O₆

- குறிகாட்டி: Starch (I₂ உடன் நீலம்; முடிவுப்புள்ளியில் நிறம் மறையும்)

- Ca²⁺, Mg²⁺ கடினத்தன்மை அளவிட

- pH 10 இல் மேற்கொள்ளப்படும்

தடையாளவீட்டு (Gravimetric Analysis)

- அவக்கேதம் வடிகட்டி, உலர்த்தி, எடை அளவீடு

- உதாரணம்: SO₄²⁻ → BaSO₄ அவக்கேதம் → எடை அளவீடு

நிறமாலைவீட்டு (Spectrophotometry)

Beer-Lambert விதி:

$$A = \varepsilon c l$$

- A = Absorbance (ஒளி உறிஞ்சல்)

- ε = Molar absorptivity (L/mol·cm)

- c = செறிவு (mol/L)

- l = path length (cm)

A = log(I₀/I)

பயன்பாடு: செறிவு-ஒளி உறிஞ்சல் வரைபட மூலம் அறியப்படாத செறிவு கண்டறிதல்
